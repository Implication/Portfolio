Superhero Fight

This was a project that I made when learning the fundamentals of node, express, and template boiler coding, hence why the site is simple in mind, my main goals of this site were to understand how to make request to an API client, how routing works, how templating works, and to render the data or results onto the browser without using a front end library such as react to keep the state of the data.

The API however has some limitations and some problems that I put some workaround for, firstly, I created my own JSON object file in order to get the names of all the superheroes, since the superhero api does not provide a list that is readily avaliable for people to use.

Secondly and what I found out later as I was making the code, the people making some of the values were extremely lazy and hard coded the value of null to certain heroes, so I Decided to replace this with N/A

This can easily be improved upon to create more routes that interact with the API, and even improve upon the results table to show the values in a better fashion, but as it is, it gets the point across.

For this project I used HTML, CSS Javascript, NODEJS, Bootstrap, Express, and EJS, all data that is shown after entering the hero names comes from the Superhero API which can be found here
